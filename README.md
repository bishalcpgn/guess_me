# Can you guess me ?


## How to Play

1. Clone or download this repository to your local machine : 

   ```bash
   git clone https://github.com/bishalcpgn/guess_me

   ```

2. Navigate to the project directory : 

   ```bash
   cd guess_me
   ```

3. Run the Hangman game :


   ```bash
   python3 guess_me.py 
   ```
